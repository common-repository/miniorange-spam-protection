SPAM Protection and Anti SPAM against comments, registrations and login. Content protection for your website.

== Description ==

A very powerful and user friendly WordPress SPAM protection and AntiSpam plugin which eliminates Comments Spam, Content Spam, Registration Spam, trackback/Pingback spam and every other type of WordPress SPAM. SPAM Protection and Anti-SPAM plugin provides security against Login, Registrations, comments SPAM and content protection for your website. It simply identify, delete and prevent future registrations and helps to defeat wordpress SPAM users.

== Installation ==

= From your WordPress dashboard =
1. Visit `Plugins > Add New`
2. Search for `Spam Protection`. Find and install the plugin.
3. Activate the plugin from your Plugins page

= From WordPress.org =
1. Download Spam Protection plugin.
2. Unzip and upload the `miniorange-spam-protection` directory to your `/wp-content/plugins/` directory.
3. Activate plugin from your wordpress dashboard.

= Once Activated =
1. Go to `Settings-> Spam Protection`, and follow the instructions.
2. Click on `Save`
